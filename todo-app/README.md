# Nest의 특징

- node.js 서버 애플리케이션을 구축하기 위한 프레임워크
- express 기반
- Typescript 사용(javascript 사용 가능)
- Nest의 요소(OOP:객체지향 프로그래밍, FP:Functional 프로그래밍, FRP:Functional React 프로그래밍)
- 외부모듈 사용에 자유로움
- unit 테스트와 e2e 테스트 툴 제공
